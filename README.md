# 📊 블랙프라이데이 소비 효과 분석
### 🔎 연구 배경
최근 지속되는 거시 경제 침체와 소비 심리 위축으로 인해 소상공인이 겪는 어려움은 커져만 가고 있습니다. \
통상적으로 다가오는 11월의 블랙프라이데이 기간은, 국내에서도 소비가 크게 증가하는 대표 할인 행사로 자리매김 했습니다.
이러한 대규모 유통 이벤트나 연휴 기간은 소비가 활성화되는 시기로 인식되며, 많은 소상공인들이 매출 부진을 타개할 **단기 매출 돌파구**로 기대하고 출혈적인 프로모션에 동참합니다.

하지만 이런 거대한 이벤트의 혜택은 대형 유통 플랫폼과 고가 품목에 집중되는 **소비 양극화** 현상을 고려할 때, 일반 소상공인에게도 실질적인 매출 반등의 기회로 작용하는지에 대한 객관적인 검증이 필요하다고 판단했습니다.

본 프로젝트는 단순한 현상 관찰을 넘어, 시계열 데이터의 내재적 패턴을 통제한 상태에서 특정 이벤트가 매출에 미치는 순수 인과효과와 통계적 유의미성을 도출하는 것을 목적으로 합니다.
